# simple-circuits

### simple-draw_circuit

##### Features:
- Draws simple circuits

##### Requirements:
- tkinter
- matplotlib
- SchemDraw (https://cdelker.bitbucket.io/SchemDraw/SchemDraw.html)
```
pip install SchedmDraw
```
